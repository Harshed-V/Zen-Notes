# 🧘 ZEN Notes

> A minimal, distraction-free writing space designed for clarity and focus.

**ZEN Notes** is a premium, lightweight note-taking application that prioritizes your thoughts. Built with a "less is more" philosophy, it provides a quiet digital environment where you can write, organize, and export your ideas without the noise of traditional productivity tools.

## 🌐 Live Demo

Experience ZEN Notes live:  
🔗 https://zen-notes-v2.vercel.app/

## ✨ Features

- **🎯 Distraction-Free UI**: A clean, minimalist interface that puts your content front and center.
- **📝 Markdown Support**: Native rendering for headers, bold/italic text, code blocks, lists, and blockquotes.
- **⚡ Autosave & Local Persistence**: Your thoughts are saved instantly to your browser's local storage—never lose a word.
- **📊 Writing Insights**: Real-time word count, character count, and estimated reading time.
- **🔍 Instant Search**: Quickly find any note with powerful real-time filtering.
- **📌 Pinning**: Keep your most important thoughts at the top of your list.
- **🌓 Adaptive Themes**: Seamless switching between a serene Light mode and a focused Dark mode.
- **📥 Markdown Export**: Download your notes as `.md` files to take them anywhere.
- **⌨️ Power-User Shortcuts**: Navigate and create at the speed of thought.

## 🛠️ Tech Stack

ZEN Notes is built using modern, high-performance web technologies:

- **Framework**: [TanStack Start](https://tanstack.com/start) (Full-stack React)
- **Runtime**: [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) (Primitives)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/zen-notes.git
   cd zen-notes
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## ⌨️ Keyboard Shortcuts

Stay in the flow with these built-in shortcuts:

| Shortcut | Action |
| :--- | :--- |
| `⌘ + N` / `Ctrl + N` | Create a new note |
| `⌘ + K` / `Ctrl + K` | Focus search bar |
| `⌘ + S` / `Ctrl + S` | Force manual save (Autosave is always on) |
| `⌘ + P` / `Ctrl + P` | Toggle Preview mode |
| `Delete` / `Backspace` | Delete active note (when not typing) |
