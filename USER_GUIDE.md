# Zen Notes - User Guide

## Getting Started

Welcome to **Zen Notes** - a beautiful, minimal note-taking application designed for focused writing.

### Creating Your First Note

1. Click the **"New"** button in the header (top right)
2. Or use the keyboard shortcut: **`Ctrl/Cmd + N`**
3. Start typing your title and content

### Writing in Markdown

Zen Notes supports full markdown formatting:

#### Headings
```
# Heading 1
## Heading 2
### Heading 3
```

#### Text Formatting
```
**bold text**
*italic text*
`inline code`
```

#### Lists
```
- Bullet item
- Another item

1. Numbered item
2. Another item
```

#### Code Blocks
````
```
code block
```
````

#### Blockquotes
```
> This is a quote
```

### Viewing Your Content

You have two modes:

#### 📝 Edit Mode
- Write and format your notes
- See raw markdown syntax
- Direct control over content

#### 👁 Preview Mode
- View fully rendered markdown
- See how your note will look
- Perfect for reviewing before sharing

**Toggle between modes** using the eye icon button in the editor toolbar.

## Features

### 🔍 Search
- Click the search box or press **`Ctrl/Cmd + K`**
- Search across all note titles and content
- Results update in real-time

### 📌 Pin Notes
- Click the pin icon to pin important notes
- Pinned notes appear at the top of your list
- Great for frequently accessed notes

### 💾 Autosave
- Your notes are automatically saved every second
- Look for the **✓ Saved** indicator in the toolbar
- All data stored locally - never sent anywhere

### ⏱ Reading Time
- Automatically calculated as you write
- Based on 200 words per minute average
- Helps you gauge content length

### 📊 Statistics
- **Word count** - Total words in your note
- **Reading time** - Estimated reading duration
- **Last edited** - When you last made changes

### 📥 Export
- Click the download icon to export as markdown
- Downloads as `.md` file with your note title
- Perfect for sharing or backup

### 🗑 Delete
- Click the trash icon to remove a note
- Removes from your library

## Data Storage

Your notes are stored in **browser local storage**. This means:

✅ **Private** - Your data stays on your device
✅ **Offline** - Works without internet connection
✅ **Fast** - Instant saves with no server delays
✅ **Persistent** - Notes survive browser refreshes

⚠️ **Important**: Clearing your browser's local storage will delete all notes. Consider exporting important notes as markdown files for backup.

## Troubleshooting

### Notes disappeared after refresh
- Check if you cleared browser local storage
- Try checking browser DevTools > Application > Local Storage
- Ensure cookies/storage isn't blocked

### Markdown not rendering
- Make sure you're in **Preview Mode** (eye icon)
- Check markdown syntax is correct
- Try editing and switching back to preview

### Notes not saving
- Check your browser allows local storage
- Ensure you're not in private/incognito mode
- Check browser storage quota
